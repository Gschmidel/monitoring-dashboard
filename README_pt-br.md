# 📊 Dashboard de Monitoramento Financeiro e Operacional

Um dashboard interativo desenvolvido para simular o monitoramento de transações financeiras e operações de sistema, permitindo identificação de riscos e tomada de decisão baseada em dados.

---

## 🚀 Visão Geral

Este projeto demonstra como a análise de dados pode ser aplicada para monitorar operações e desempenho financeiro em um ambiente de negócios.

Ele oferece:

- Monitoramento de transações e erros  
- Classificação de risco  
- KPIs para tomada de decisão  
- Filtros interativos (data e contexto)

---

## 🎯 Problema de Negócio

Empresas enfrentam desafios como:

- Identificar transações de alto risco  
- Monitorar falhas operacionais  
- Priorizar problemas com base em impacto  
- Tomar decisões rápidas baseadas em dados  

---

## 💡 Solução

O dashboard permite alternar entre:

### 🔹 Financeiro
- Volume de transações ao longo do tempo  
- Distribuição de risco  
- Categorias com maior receita  

### 🔹 Operacional
- Evolução de erros  
- Severidade das falhas  
- Módulos com mais problemas  

---

## 📊 Funcionalidades

- Dashboard interativo com Dash  
- KPIs em destaque  
- Filtro por data  
- Layout estilo BI (com sidebar)  
- Ranking Top 5
